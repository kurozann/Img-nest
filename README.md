# Img-nest

- A lightweight, open-source repository for organizing and storing categorized images.
- Primarily focused on anime-style characters, cosplay visuals, and aesthetic references.
- Designed as a flexible "nest" for creators, developers, and enthusiasts who work with visual datasets or character concepts.

## • Features
- Clean folder structure with theme-based categories
- Supports high-resolution images
- Ideal for inspiration, AI datasets, and reference work
- Public and open-source — free to fork or contribute

## • Folder Structure
- `blue-archive/` – Visuals from Blue Archive characters
- `cosplay/` – Cosplay-style references and poses
- `misc/` – A JSON file containing URL and metadata of each image

## • Usage
- Use images for creative projects, character design, or machine learning references
- Contributions are welcome — fork the repo and open a pull request
- Ensure content is safe, legal, and credits are respected where applicable
